# PCB SMT 制程与 AOI 检测

## SMT 表面贴装技术简介

SMT（Surface Mount Technology）是现代 PCB 组装的主流工艺。与传统插件（Through-Hole）不同，SMT 元件直接贴装在 PCB 表面，无需钻孔插脚。

核心流程：锡膏印刷 → 贴片 → 回流焊 → AOI 检测 → 返修

## AOI 自动光学检测

AOI（Automated Optical Inspection）利用高分辨率相机拍摄 PCB 图像，通过图像处理和深度学习算法自动识别缺陷。

传统 AOI 局限：
1. 依赖人工设定检测规则，灵活性差
2. 对微小缺陷、异形缺陷检测率低
3. 误报率高，需要大量人工复判

YOLO 深度学习方案优势：
1. 端到端检测，无需手动设计特征
2. 对变形、光照变化鲁棒性强
3. 支持多类别并行检测
4. 可在线学习，持续提升检测精度

## 检测参数调优

### 置信度阈值（Confidence Threshold）
- 默认 0.25，可根据产线要求调整
- 提高阈值：减少误检，但可能漏检
- 降低阈值：提高召回，但误报增多
- 建议：初期 0.25，根据复判反馈逐步优化

### IoU 阈值（Intersection over Union）
- 默认 0.45，用于 NMS 去重
- 提高 IoU：保留更多检测框，可能重复
- 降低 IoU：去重更激进，可能漏掉相邻目标
- 建议：保持默认 0.45

### 输入分辨率（Image Size）
- 640x640 为标准分辨率
- 更高的分辨率可检测更小的缺陷，但推理速度下降
- CPU 模式建议 416x416 以提升速度

## PCB 检测场景配置

SPRIDS 平台支持针对不同 PCB 型号创建独立检测场景，每个场景可绑定：
1. 专属缺陷类型字典
2. 定制化 YOLO 模型权重
3. 差异化检测参数
4. 独立的历史统计和趋势分析

建议为每条 SMT 产线创建独立场景，便于品质追踪和模型迭代。
