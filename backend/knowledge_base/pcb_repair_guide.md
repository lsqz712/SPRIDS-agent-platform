# PCB 缺陷维修指南

## 短路维修

1. 使用吸锡带清除多余焊锡
2. 检查钢网开口是否过大——开口面积应 ≤ 焊盘面积的 90%
3. 调整回流焊峰值温度，降低至 235-240°C
4. 若为设计问题，增加相邻导线间距至 ≥ 0.15mm

## 开路维修

1. 使用飞线或导电银浆修补断裂线路
2. 检查元件引脚是否氧化——用万用表测量接触电阻
3. 确认焊膏印刷量是否充足
4. 检查回流焊保温时间，确保焊膏充分熔化

## 缺孔处理

1. 确认钻孔文件版本是否正确
2. 检查钻头寿命管理记录
3. 对已生产的缺陷板，使用手动钻孔或跳线方案
4. 建立钻孔后 AOI 检验工序

## 鼠咬修复

1. 轻微鼠咬可涂覆绝缘漆保护
2. 严重缺口需报废处理
3. 调整分板机铣刀速度至推荐值
4. 优化邮票孔数量和分布，推荐 5 孔连接

## 毛刺处理

1. 轻微毛刺可用细砂纸打磨去除
2. 检查蚀刻液浓度和温度是否在工艺范围内
3. 增加曝光能量 5-10%
4. 优化显影液喷淋压力

## 残铜清除

1. 使用刻刀或激光去除孤立残铜
2. 检查干膜贴附质量——贴附后需静置 15 分钟
3. 确认曝光机的真空度是否达标
4. 建立显影后目检工序

## 批次质量追踪

SPRIDS 平台支持按 PCB 批次统计良品率：
- 批次良品率 = 良品数 / 检测总数 × 100%
- 目标良品率：≥ 98%
- 当良品率 < 95% 时触发预警
- 按缺陷类型追踪分布趋势，定位工艺瓶颈
