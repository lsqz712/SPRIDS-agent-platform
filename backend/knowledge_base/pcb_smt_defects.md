# PCB SMT 常见缺陷类型

## 缺孔（Missing Hole）

缺孔是指 PCB 上缺少应有的孔位。在 SMT 生产过程中，钻孔工序遗漏或钻孔文件错误都会导致缺孔。

原因：
1. 钻孔程序遗漏该孔位坐标
2. 钻头折断后未及时更换
3. 钻孔文件版本错误

影响：元件无法插装，电路无法导通，属于严重缺陷。

## 鼠咬（Mouse Bite）

鼠咬指 PCB 边缘出现类似鼠咬的锯齿状缺口。通常发生在拼板（panelization）的 V-Cut 或邮票孔连接处。

原因：
1. 分板机参数设置不当，铣刀切入过深
2. V-Cut 深度过大导致分板时撕裂
3. 邮票孔布局不合理

影响：可能影响 PCB 结构强度，严重时导致线路断裂。

## 开路（Open Circuit）

开路是指导电线路出现断裂，电流无法流通。这是 PCB 最致命的缺陷之一。

原因：
1. 蚀刻过度导致线路过细或断裂
2. 钻孔偏移导致过孔断开
3. 焊接不良导致焊点开路
4. 机械应力导致铜箔断裂

检测方法：AOI 可检测线路完整性，飞针测试可验证导通性。

## 短路（Short Circuit）

短路是指不应连接的两条导线之间出现意外导通。短路会导致电路功能异常甚至烧毁元件。

原因：
1. 蚀刻不足导致铜箔残留桥接
2. 焊锡过量导致焊盘桥连
3. 元件引脚弯曲接触相邻焊盘
4. PCB 设计间距过小

修复建议：
1. 检查钢网开口尺寸，减小焊锡量
2. 调整回流焊温度曲线
3. 优化 PCB 布线间距

## 毛刺（Spur）

毛刺是蚀刻残留导致的铜箔尖刺状突起。通常出现在线路边缘或焊盘周围。

原因：
1. 蚀刻液浓度不均匀
2. 曝光时间控制不当
3. 显影不充分

影响：可能导致阻抗不匹配，高速信号反射。严重时毛刺断裂形成短路隐患。

## 残铜（Spurious Copper）

残铜指 PCB 表面本应被蚀刻去除但残留的多余铜箔。与毛刺不同，残铜通常是孤立的岛状区域。

原因：
1. 干膜附着不良导致蚀刻液渗入
2. 曝光能量不足导致干膜未完全固化
3. 显影过度导致保护膜脱落

影响：可能形成意外导电路径，降低 PCB 可靠性。
