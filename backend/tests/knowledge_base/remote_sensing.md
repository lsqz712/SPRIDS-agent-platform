# 遥感目标检测知识

## 遥感图像特点

遥感图像是从卫星或航空器上获取的地球表面图像，具有以下特点：
1. **高分辨率**：现代遥感卫星可达 0.3~1m 分辨率
2. **大幅面**：单张图像通常覆盖数十平方公里
3. **多光谱**：除可见光外，还包含近红外、短波红外等波段
4. **目标尺度变化大**：同一图像中可能同时存在小型车辆和大型建筑物

## 常见遥感目标类别

本项目检测 4 类遥感目标：
1. **飞机（airplane）**：通常停在跑道或停机坪上，形状规则，尺寸较大
2. **油罐（storage-tank）**：圆形或圆柱形结构，通常成组出现在储油区
3. **立交桥（overpass）**：多层道路交叉结构，呈立体交叉形态
4. **操场（baseball-diamond）**：矩形场地，通常有跑道环绕

## 遥感目标检测挑战

1. **尺度变化**：同一图像中目标尺寸差异可达数十倍
2. **背景复杂**：自然背景、人造建筑、植被等混合
3. **目标密集**：某些区域目标分布密集，相互遮挡
4. **光照变化**：不同时间、天气条件下拍摄的图像差异大

## 数据增强策略

针对遥感图像的特点，常用的数据增强方法：
1. **随机裁剪**：模拟不同分辨率下的目标观测
2. **旋转翻转**：增强模型对目标方向的鲁棒性
3. **亮度对比度调整**：模拟不同光照条件
4. **马赛克增强**：将多张图像拼接，增加小目标数量

## 评价指标

遥感目标检测通常使用以下指标：
- **mAP50**：IoU 阈值 0.5 时的平均精度
- **mAP50-95**：多阈值平均精度，更严格
- **Recall**：召回率，衡量漏检情况
- **FPS**：每秒帧率，衡量实时性
