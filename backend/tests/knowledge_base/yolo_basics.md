# YOLO 目标检测基础知识

## 什么是 YOLO

YOLO（You Only Look Once）是一种端到端的实时目标检测算法。与传统的两阶段检测方法（如 R-CNN）不同，YOLO 将目标检测视为一个回归问题，在一次前向传播中同时预测边界框和类别概率。

## YOLOv11 的改进

YOLOv11 是 Ultralytics 发布的最新一代 YOLO 模型，主要改进包括：
1. **更高效的特征提取**：采用改进的 C3k2 模块，增强多尺度特征融合
2. **注意力机制**：引入 C2PSA 模块（Spatial Attention），提升对小目标的检测能力
3. **更灵活的架构**：支持 n/s/m/l/x 五种规模，适配不同算力场景
4. **训练策略优化**：改进的数据增强和学习率调度策略

## YOLO11n 模型

YOLO11n 是 YOLOv11 系列中最轻量的版本：
- 参数量：约 2.6M
- 推理速度：CPU 约 50ms/张，GPU 约 5ms/张
- 适用场景：边缘设备部署、实时检测、资源受限环境
- 精度：mAP50 约 0.6-0.7（在 COCO 数据集上）

## 什么是置信度（Confidence）

置信度是模型对检测结果的确信程度，取值范围 0~1：
- 置信度 > 0.9：模型非常确信，检测结果高度可信
- 置信度 0.5~0.9：模型较为确信，一般可信
- 置信度 < 0.5：模型不太确信，可能是误检

设置置信度阈值（conf threshold）可以过滤低质量的检测结果。阈值越高，检测越严格，漏检可能增多；阈值越低，检测越宽松，误检可能增多。

## 什么是 IoU（Intersection over Union）

IoU（交并比）是衡量两个边界框重叠程度的指标：

IoU = 交集面积 / 并集面积

- IoU = 1：两个框完全重合
- IoU = 0：两个框完全不重叠
- IoU > 0.5：通常认为检测正确

IoU 在目标检测中的两个用途：
1. **评估指标**：判断预测框与真实框的匹配程度
2. **NMS（非极大值抑制）**：去除重叠度高的重复检测框

## 什么是 NMS（非极大值抑制）

NMS（Non-Maximum Suppression）是目标检测后处理的关键步骤，用于去除对同一目标的重复检测。

工作流程：
1. 按置信度从高到低排序所有检测框
2. 选择置信度最高的框，保留
3. 计算其余框与已选框的 IoU
4. 删除 IoU 大于阈值的框（认为是同一目标的重复检测）
5. 重复步骤 2-4，直到处理完所有框

## 什么是 mAP（mean Average Precision）

mAP 是目标检测最常用的评估指标：
- **Precision（精确率）**= TP / (TP + FP)，检测出的目标中有多少是正确的
- **Recall（召回率）**= TP / (TP + FN)，所有真实目标中有多少被检测出来
- **AP（Average Precision）**= Precision-Recall 曲线下的面积
- **mAP**= 所有类别的 AP 的平均值

常见的 mAP 变体：
- mAP50：IoU 阈值为 0.5 时的 mAP
- mAP50-95：IoU 阈值从 0.5 到 0.95（步长 0.05）的平均 mAP，更严格
